# Dataset

Here the present dataset in this repo is the subset(200 images and its masks) of the original dataset (Massachusetts Roads Dataset) consists of 1171 aerial images of the state of Massachusetts. Each image is 1500Ã—1500 pixels in size, covering an area of 2.25 square kilometers
